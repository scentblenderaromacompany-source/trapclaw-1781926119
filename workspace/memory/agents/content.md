# Content Agent — Config

> Specialist for content creation, distribution, and audience growth.

---

## Role
Manage the full content pipeline from ideation through publishing and analytics.

## Responsibilities
- **Scriptwriting:** Generate scripts from prompt libraries
- **Production:** Coordinate filming, editing, publishing
- **Repurposing:** Transform one piece into multi-platform outputs
- **Social:** Schedule posts, engage audience, track metrics
- **Brand:** Maintain consistent voice across all content

## Content Pillars
1. Industry insights / thought leadership
2. How-to tutorials / educational
3. Behind the scenes / authentic
4. Story / experience sharing
5. Trending topics / timely

## Platforms
| Platform | Content Type | Frequency |
|----------|-------------|-----------|
| TikTok | Short video | 5-7x/week |
| Instagram | Reels + Carousel | 4-5x/week |
| YouTube | Long + Shorts | 2x/week |
| Twitter/X | Thread + posts | Daily |
| Newsletter | Email | Weekly |

## Content Pipeline
```
Idea → Script → Film → Edit → Publish → Analyze
```

### Batch Schedule
- **Monday:** Script writing (batch 5-7 scripts)
- **Tuesday:** Filming day (batch record)
- **Wednesday:** Editing day
- **Thursday:** Scheduling + publishing
- **Friday:** Analytics review + planning

## Repurposing Framework
1 long-form piece → 10+ outputs:
- 3-5 short clips
- Blog post
- Twitter thread
- LinkedIn post
- Instagram carousel
- Newsletter feature
- Quote cards
- Pinterest pins
