# Video Production SOP

## Purpose
Standardized workflow for producing short-form and long-form video content across platforms.

## Pre-Production

### Content Planning
1. Review content calendar (weekly batch preferred)
2. Select topic from [[Video Scripts]] library
3. Write/adapt script
4. Gather B-roll and assets
5. Check trending audio on each platform

### Equipment Checklist
- [ ] Camera (phone or DSLR) charged
- [ ] Microphone connected and tested
- [ ] Lighting set up (ring light minimum)
- [ ] Tripod/mount stable
- [ ] Clean background

## Production

### Shot Types for Short-Form
| Type | Duration | When to Use |
|------|----------|-------------|
| Hook | 0-3 sec | Every video — pattern interrupt |
| Talking head | 15-30 sec | Main content delivery |
| B-roll | 3-5 sec | Transitions, visual interest |
| Text overlay | varies | Key points, captions |
| CTA | 3-5 sec | End of video |

### Filming Rules
- Film horizontal for YouTube, vertical for TikTok/Reels/Shorts
- 4K preferred, 1080p minimum
- Record 10 sec of silence at start (room tone)
- Film 2 takes minimum per segment
- Check audio levels before each batch

## Post-Production

### Editing Workflow
1. Import and organize footage
2. Rough cut — arrange segments per script
3. Add transitions (keep simple — jump cuts work)
4. Text overlays and captions
5. Music/sound effects
6. Color correction
7. Export: 1080x1920 (vertical), 1080x1080 (square), 1920x1080 (horizontal)

### Caption/Subtitle Rules
- Always add captions (85% of social video watched muted)
- Font: bold, high contrast, centered bottom third
- Max 2 lines on screen at once
- Auto-sync then manually correct

## Publishing

| Platform | Format | Max Length | Best Time |
|----------|--------|------------|-----------|
| TikTok | 9:16 | 10 min | 7-9 AM, 12-2 PM, 7-10 PM |
| Instagram Reels | 9:16 | 90 sec | 11 AM-1 PM, 7-9 PM |
| YouTube Shorts | 9:16 | 60 sec | 12-3 PM |
| YouTube Long | 16:9 | 12+ min | 2-4 PM |
| Twitter/X | 16:9 or 1:1 | 2:20 | 8-10 AM, 12-1 PM |

### Post-Publish Checklist
- [ ] Title optimized with keywords
- [ ] Description includes hashtags (3-5 relevant)
- [ ] Thumbnail uploaded (YouTube)
- [ ] Cross-post to other platforms
- [ ] Log in content tracker

## Metrics to Track
- View count (24h, 7d, 30d)
- Watch time / retention curve
- Engagement rate (likes + comments + shares / views)
- Follower growth attributed
- Click-through rate (links in bio)

## Batch Production Schedule
- **Monday:** Script writing (batch 5-7 scripts)
- **Tuesday:** Filming day (batch record)
- **Wednesday:** Editing day
- **Thursday:** Scheduling and publishing
- **Friday:** Analytics review + next week planning
