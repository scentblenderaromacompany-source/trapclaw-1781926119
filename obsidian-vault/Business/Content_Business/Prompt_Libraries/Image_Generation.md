# Image Generation Prompts

## Purpose
Reusable prompt library for AI image generation (Midjourney, DALL-E, Stable Diffusion, Flux).

## Prompt Structure Formula
```
[Subject] + [Style] + [Lighting] + [Composition] + [Color Palette] + [Technical Specs]
```

## Product Photography Prompts

### Jewelry - Hero Shot
```
Product photography of [item], placed on [surface], 
soft diffused studio lighting, white background, 
sharp focus, 8K, commercial quality, 
shot on Phase One IQ4 150MP --ar 1:1 --s 750
```

### Jewelry - Lifestyle
```
Elegant woman wearing [item], [setting description], 
golden hour natural lighting, shallow depth of field, 
editorial fashion photography, Vogue aesthetic, 
warm tones --ar 4:5 --s 800
```

### Jewelry - Flat Lay
```
Flat lay of [items] on [marble/linen/wood], 
overhead shot, minimal styling, soft shadows, 
clean aesthetic, Pinterest-worthy, 
natural light --ar 1:1 --s 600
```

## Social Media Graphics

### Quote Card
```
Minimalist quote card, "[quote text]", 
modern sans-serif typography, [color] background, 
clean design, Instagram ready --ar 1:1 --s 500
```

### YouTube Thumbnail
```
YouTube thumbnail, [subject], expressive face/reaction, 
bold text "[title]", vibrant colors, 
high contrast, click-worthy, 
professional --ar 16:9 --s 700
```

### Carousel Slide
```
Educational infographic slide, [topic], 
clean layout, [color scheme], 
icons and illustrations, modern flat design, 
professional presentation --ar 1:1 --s 600
```

## Brand Consistency

### Color Palettes
- **Jewelry Store:** Gold (#C9A84C), Cream (#F5F0E8), Charcoal (#2D2D2D), White (#FFFFFF)
- **Content Brand:** [Define brand colors here]
- **Neutral/Universal:** Soft whites, warm grays, natural tones

### Style Keywords to Always Include
- "professional," "high quality," "clean," "modern"
- Avoid: "cartoon," "illustration" (unless intended)

## Negative Prompts (Universal)
```
ugly, blurry, low quality, text errors, watermark, 
deformed, disfigured, bad anatomy, extra limbs, 
oversaturated, amateur, stock photo
```
