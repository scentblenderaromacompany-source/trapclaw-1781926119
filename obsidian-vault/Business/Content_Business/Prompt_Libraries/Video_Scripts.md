# Video Script Prompts

## Purpose
Reusable script templates and AI prompt frameworks for generating video content.

## Hook Templates (First 3 Seconds)

### Curiosity Hook
```
"Most people don't know this about [topic], 
and it's costing them [consequence]..."
```

### Controversy Hook
```
"Unpopular opinion: [bold statement]. 
Here's why..."
```

### Story Hook
```
"So I was [situational context] when [unexpected event]...
and that's when I realized [insight]."
```

### Stats Hook
```
"[Shocking statistic]. Let me explain why 
this matters for [audience]..."
```

### Challenge Hook
```
"I tried [thing] for [time period]. 
The results were [unexpected/emotional]..."
```

## Short-Form Script Templates

### Educational (30-60 sec)
```
[HOOK] — 0-3 sec
[PROBLEM] — 3-8 sec (what viewers struggle with)
[SOLUTION] — 8-25 sec (the insight/fix)
[PROOF/EXAMPLE] — 25-40 sec (evidence)
[CTA] — 40-45 sec (follow, comment, link in bio)
```

### Storytime (60-90 sec)
```
[HOOK] — "Let me tell you about [event]..."
[RISING ACTION] — build tension
[CLIMAX] — the turning point
[RESOLUTION] — what happened
[LESSON] — what you learned
[CTA] — "Follow for more stories like this"
```

### Listicle (45-60 sec)
```
[HOOK] — "Here are [N] [things] you need to know about [topic]"
[ITEM 1] — with brief explanation
[ITEM 2] — with brief explanation
[ITEM 3] — with brief explanation
[BONUS] — surprise extra
[CTA] — "Which one surprised you? Comment below"
```

## Long-Form Script Template (8-15 min YouTube)

```
[INTRO — 0:00-0:30]
  - Hook + channel branding
  - Preview of what they'll learn
  - "Stay until the end for [bonus]"

[CONTEXT — 0:30-2:00]
  - Why this topic matters
  - Personal connection/experience
  - What viewers will be able to do after

[MAIN CONTENT — 2:00-10:00]
  Section 1: [Topic]
    - Explanation
    - Example
    - Visual/B-roll
  Section 2: [Topic]
    - Explanation  
    - Example
    - Visual/B-roll
  Section 3: [Topic]
    - Explanation
    - Example
    - Visual/B-roll

[SYNTHESIS — 10:00-12:00]
  - Bring it all together
  - Key takeaways (numbered)

[CTA — 12:00-13:00]
  - Subscribe reminder
  - Like if valuable
  - Comment question
  - Next video teaser
```

## AI Script Generation Prompts

### Generate Short-Form Script
```
Write a 45-second TikTok script about [topic].
Target audience: [audience]
Tone: [casual/professional/funny]
Include: strong hook, 3 key points, CTA
Format: timestamped sections
```

### Generate Long-Form Outline
```
Create a detailed YouTube video outline for [topic].
Target: [audience]
Length: 10-12 minutes
Include: timestamps, B-roll suggestions, 
engagement hooks every 2-3 minutes, 
thumbnail concept, title options (5)
```

### Adapt Script for Platform
```
Take this YouTube script and adapt it for [TikTok/Reels/Shorts].
Keep the core message, shorten to [time], 
make the hook more aggressive, 
add platform-specific CTA.
```
