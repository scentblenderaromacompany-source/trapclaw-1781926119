# Jewelry Listing SOP

## Purpose
Standardize the process of creating product listings across all sales channels (eBay, Etsy, Shopify, Poshmark).

## Pre-Listing Checklist
- [ ] Item cleaned and polished
- [ ] Hallmarks/stampings photographed and documented
- [ ] Weight recorded (grams)
- [ ] Measurements taken (length, width, stone sizes)
- [ ] Condition graded: New / Like New / Excellent / Good / Fair
- [ ] Material identified (gold karat, silver type, platinum, etc.)
- [ ] Stones identified (natural vs synthetic, treatment status)

## Photography Requirements
→ See [[Photography]] for detailed photo SOP

Minimum shots per listing:
1. Front view (white background)
2. Back view
3. Close-up of hallmarks/stamps
4. Scale reference (next to coin or ruler)
5. Worn/on-body shot (if available)

## Listing Template

```
Title: [Material] [Type] [Style] [Era/Period if notable] — [Key Feature]
Example: 14K Gold Art Deco Diamond Ring — 0.5ct Center Stone

Description Structure:
1. Opening hook (1 sentence)
2. Key specs (material, weight, dimensions)
3. Stone details
4. Condition notes
5. History/provenance (if known)
6. Care instructions
7. Return policy
```

## Channel-Specific Notes

### eBay
- Use all 12 photo slots
- Item specifics: fill every applicable field
- Free shipping baked into price
- 30-day returns

### Etsy
- Tags: use all 13
- Focus on handmade/vintage narrative
- Sustainability angle

### Shopify
- SEO-optimized descriptions
- Cross-sell recommendations
- Email capture on product pages

## After Listing
- [ ] Verify listing is live and correct
- [ ] Log in inventory spreadsheet
- [ ] Set 7-day reminder to check views/favorites
- [ ] Adjust pricing after 14 days if no engagement
